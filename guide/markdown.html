Markdown 语法全面参考 ​

Markdown 是一种轻量级标记语言，下面是其常用语法总结。

标题

Markdown 使用 # 来表示标题。# 的个数决定了标题的级别，从 # (一级标题) 到 ###### (六级标题)。

markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

段落 段落是由空行分隔的文本块。

markdown
这是一个段落。

这是另一个段落。

强调

Markdown

斜体：使用一个星号 * 或者下划线 _ 包裹文本。

粗体：使用两个星号 ** 或者下划线 __ 包裹文本。

粗斜体：使用三个星号 *** 包裹文本。

粗斜体

markdown
*斜体* 或者 _斜体_

**粗体** 或者 __粗体__

***粗斜体***

列表

无序列表,使用星号 *、加号 + 或者减号, - 来创建无序列表。

markdown

- 项目 1
- 项目 2
- 项目 3

或者

+ 项目 1
+ 项目 2
+ 项目 3

有序列表
使用数字加点 1. 来创建有序列表。


1. 第一点
2. 第二点
3. 第三点

嵌套列表,通过缩进来创建嵌套列表。

1. 第一项
    - 子项 1
    - 子项 2
2. 第二项

链接

行内链接

markdown

[百度](https://www.baidu.com)

锚点链接

[回到顶部](#标题)

自动链接
自动识别网址和电子邮件地址。

http://www.example.com
example@example.com

图片语法与链接相似，前面加一个 !。

![替代文本](https://www.example.com/image.jpg)

引用 使用 > 来创建引用。

markdown

> 这是一个引用。

多级引用

> 这是第一层引用
> > 这是第二层引用

代码

行内代码,使用反引号 ` 包裹代码。

markdown

这是 `代码` 示例。

多行代码块
使用三个反引号 ``` 或者四个空格进行缩进。

<pre> ``` 多行代码块 可以包含多行内容 ``` </pre>

语法高亮

可以通过指定语言来启用语法高亮。

markdown
```python
print("Hello, World!")

表格

使用竖线 | 和减号 - 来创建表格。

markdown
| 标题 1 | 标题 2 | 标题 3 |
| ------ | ------ | ------ |
| 内容 1 | 内容 2 | 内容 3 |
| 内容 A | 内容 B | 内容 C |

水平线

使用三个或更多的短横线 -、星号 * 或下划线 _ 来创建水平线。

markdown

---
***
___

任务列表

使用 [ ] 来创建任务列表。

markdown

- [ ] 任务 1
- [x] 任务 2（已完成）
- [ ] 任务 3
